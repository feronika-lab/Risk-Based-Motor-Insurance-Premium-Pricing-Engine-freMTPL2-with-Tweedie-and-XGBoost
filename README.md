# Risk-Based-Motor-Insurance-Premium-Pricing-Engine-freMTPL2-with-Tweedie-and-XGBoost
Membangun risk-based motor insurance premium pricing engine dari data freMTPL2 untuk memprediksi pure premium. Pipeline end-to-end: data prep, Tweedie GLM (actuarial) dan XGBoost (ML), evaluasi risk ranking via Normalized Gini, lalu premium calculator.
